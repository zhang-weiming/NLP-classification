# 原始数据集目录


## 文件包括

写代码时用的是kaggle真假新闻数据集，解压后是news.csv文件。
下载地址：https://www.kaggle.com/nopdev/real-and-fake-news-dataset


## 内容格式
    
|   label  |   text  |
| :----:   | :----:  |
| o  | txtxtxtxt |
| ...  | ... |
| 1  | txtxtxtxt |
| ...  | ... |


## 字段含义
- label: 类别id，0表示真、1表示假，或者反过来。
- text: 文档内容。
